В коде описаны несколько моделей

В файле construction.cc:
Модель в функции Cherenkov описывает модуль антинейтринного детектора, в котором есть:
	-Сцинтиллятор и сборка из них(любого количества сцинтилляторов)
	-Нейтронный поглотитель и сборка из них(для любого количества сцинтилляторов наносится на длинные торцы)
	-калибратор для различных частиц
	-оптические фибры
	-ФЭУ под размер меньшего торца сцинтиллятора;
В функции Scintillator на данный момент рассматривается гранула B2O3 и проводится 
моделирование параметра доли вылетаемых alfa-частиц и ионом лития из гранулы.

В файле generator.cc:
Возможен запуск частиц из конкретной позиции с конкретным моментом, в произвольном
направлении в 4pi, а так в конус произвольного телесного угла(конусный калибратор)

Прилагаются расчеты:
	-В файле capteffneutron.root записаны данные по количеству захваченных нейтронов в борном поглотителе, то есть рассчитана эффективность захвата(77%)
	-В файле fibropt.root доля зарегистрированных ФЭУ фотонов, разыгранных
	в центре фибра.(33%)	
	
Далее планируется улучшение модели, рассмотрение гранульной структуры борного 
поглотителя, для добавления его в итоговую модель модуля детектора.	

Также добавлены еще несколько расчетных данных в репизиторию
